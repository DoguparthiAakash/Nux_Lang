# Nux Language Support

This extension provides comprehensive language support for the **Nux** programming language in Visual Studio Code.

## Features

- **Syntax Highlighting**: Rich coloring and formatting for Nux syntax (`.nux` and `.nuxc` files).
- **File Icons**: Custom, distinct document icons for `.nux` source files and `.nuxc` compiled binaries.
- **Language Configurations**: Native commenting, bracket matching, and auto-closing pairs for a seamless coding experience in Nux.

## Installation

1. Search for "Nux Language" in the VS Code Extensions panel, or download the `.vsix` file.
2. If using the `.vsix` file, run:
   ```bash
   code --install-extension nux-lang-0.0.1.vsix
   ```
3. Reload your VS Code window.
